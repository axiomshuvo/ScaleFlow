# ScaleFlow - React + DaisyUI Landing Page

This project is a React + Vite landing page inspired by the provided marketing design screenshot.

## Setup

1. `npm install`
2. `npm run dev`

## Build

- `npm run build`
- `npm run preview`

## Features

- Tailwind CSS + DaisyUI
- Hero + About + Services + Pricing + FAQ + Testimonials + Footer
- Responsive layout
